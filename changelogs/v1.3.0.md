## Changelog v1.3.0

## Features:
- Next Flight integration — a new sidebar card and dedicated Next Flight page show details for your booked flight, with graceful handling when no flight is booked or the livery is missing from the catalog
- "Not installed" view added to the Liveries filter so you can quickly see what's still missing
- Sidebar now shows live counts next to Liveries and Packages
- Refreshed Updates icon in the sidebar
- Updated underlying frameworks (Electron 42.1, TypeScript 6, Vite 8.0.13, React, react-router, and more)

### Bug Fixes:
- Fixed the Liveries list jumping back to the first page unexpectedly (#63)
- Improved installed livery card titles and description formatting (#64)
- Fixed the update badge layout when the sidebar is collapsed (#61)
- Fixed cards being clipped on the Liveries, Downloads, and Next Flight pages
